NTN Chapter 8: Forms

    Forms
  • Forms are made up of a form element that contains form controls such as input fields, select menus and buttons
  • Traditionally, when a form is submitted the information would be sent to a server where the information would be processed using a 'back end' language such as PHP or Ruby.
  • It's becoming more possible to process the information in a form on the 'front end' before it is sent to the server.
