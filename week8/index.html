For Real Life Chapter 8: Transforms and Transitions

    Transforms
  • The CSS3 transform property lets you translate, rotate, scale, and/or skew any elements on the page.
  • You can manipulate an element's appearance using transform function. The value of the transform property is one of more transform functions (separated by spaces) that will be applied in the order they're provided.
  • TRANSLATION: is a function that allows you to move elements left, right, up or down. It is similar to the behavior of position:relative; when declaring top and left, moving elements up and down or left and right along the x and y aces.
  • the translace(x,y) function moves and element "x" from the left and "y" from the top.
    EXAMPLE: transform: translate(45px, -45px);
  • If you only want to move an element vertically or horizontally you can use the translateX or translateY functions respecticaly.
    EXAMPLE: if you want to move 45px to the right along the x axis: transform: translateX(45px);
    to move up along the y axis by 30px, include: transofrm: translateY(-30px);
  • SCALING: the scale(x,y) function scales an element by the defined factors horizontally then vertically. If only one value is provided it will be used for both the x and y values, growing or shrinking your element or pseudo-element while maintaining the original aspect ratio.
  • As with translate, you can also use the scaleX(x) or scaleY(y) functions.
    EXAMPLE: transform: translateX(40px) scale(1.5);
  • ROTATION: the rotate() function rotates an element around the point of origin by a specified angle value.
    EXAMPLE: transform: rotate(10deg) translateX(40px) scale(1.5);
  • SKEW: the skew(x,y) function specifies a skew along the x and y axes.
    Transitions
  • Transitions allow the values of CSS properties to change over time, essentially providing simple animations.
  • Here are the steps to create a simple transition using only CSS:
    1. Declare the original state of the element in the default style declaration.
    2. Declare the final state of your transitioned element; for example, a :hover state.
    3. Include the transition functions in your default style declaration using the transition properties, including: transition-property, transition-duration, transition-timing-function, and transition-delay. We’ll look at each of these and how they work shortly.
  • The transition-property property defines the CSS properties of the element that should be transitioned, with all for all properties being the default.
  • The transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state.
  • The transition-timing-function lets you control the pace of the transition in even more granular detail.
  • The most common timing functions include the key terms ease, linear, ease-in, ease-out, or ease-in-out.
  • You can comgine all the transition values into a shorthand declaration:
    EXAMPLE: transition: transform 0.2s ease-out 50ms;}
    ANOTHER EXAMPLE: transition: transform 0.2s ease-out 50ms, color 0.1s linear 50ms;
    Animations
  • CSS animations, unlike transitions, allow you to control each step of an animation via keyframes.
  • To animate an element in CSS, you first create a named animation, then attach it to an element in that element’s property declaration block. Animations in themselves don’t do anything; in order to animate an element, you’ll need to associate the animation with that element.
  • Here are a few simple animations:
    @keyframes moveRight { from { transform: translateX(-50%); } to { transform: translateX(50%); } }
    @keyframes appearDisappear { 0%, 100% { opacity: 0; } 20%, 80% { opacity: 1; } }
    @keyframes bgMove { 100% { background-position: 120% 0; } }
  • animation-name: This property is used to attach an animation (previously defined using the @keyframes syntax) to an element:
    animation-name: appearDisappear;
  • animation-duration: he animation-duration property defines the length of time (in seconds or milliseconds) an animation takes to complete one iteration (all the way through, from 0% to 100%):
    animation-duration: 300ms;
  • animation-delay: The animation-delay property is used to define how many milliseconds or seconds to wait before the browser begins the animation:
    animation-delay: 50ms;

For Real Life Chapter 12: Canvas, SVG, and Drag and Drop

    Canvas
  • Since drawing on the canvas is done using JavaScript, we’ll need a way to grab the element from the DOM. We’ll do so by giving our canvas an ID:
    canvas id="myCanvas" class="myCanvas"
  • Drawing a rectangle to the Canvas:
    var canvas = document.getElementById("myCanvas");
    var context = canvas.getContext("2d");
    context.strokeStyle = "red";
    context.fillStyle = "rgba(0, 0, 255, 0.5)";
    context.fillRect(10, 10, 100, 100);
    context.strokeRect(10, 10, 100, 100);
