This Week's Activity: Story Editor

Notes for this week

  • Computers are stupid, even though they seem smart.
  • Any information stored or shared with a user is called OUTPUT.
  • Anything coming from where it was stored or anything the user enters is called INPUT.
  • The section tag fills the screen all the way across
  • In JS, in order to get the user’s input from any input or text area tag, you use the special word ‘value’.
  • When you want to show something to a user, you use a section and the special word ‘innerHTML’.
  • Input or textarea USE value
  • Section USE innerHTML
